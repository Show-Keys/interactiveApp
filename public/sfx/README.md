Drop your sound effect files in this folder.

Background music (optional):
- space-chords-loop.mp3

Intro (optional):
- cinematic-intro.mp3 (about 11 seconds)

Default expected filenames:
- tap.mp3
- open.mp3
- close.mp3
- select.mp3

You can use .wav too, just rename accordingly (or update the mapping in src/utils/sfx.ts).
